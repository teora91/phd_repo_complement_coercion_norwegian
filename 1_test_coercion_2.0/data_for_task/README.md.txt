## Data Files

- **llm_list_nor.json** – List of language models grouped by category to enable efficient batch evaluation during experiments.

- **entity_verb_association_updated_02-12-24.xlsx** – Resource linking entities to their typical associated verbs used in this case to extract lemmatized entities

- **list_lemma_coercion_inflected_updated_02-12-24.xlsx** – List of entities in definite inflected forms.

- **experiment_coercion_log_meto_with_context.xlsx** – Dataset containing the complement coercion experimental items with contextualized sentences used for model evaluation.
